# 🏫 Rayalaseema High School, Kalikiri

A premium, modern educational landing page designed to showcase excellence in education, discipline, and values. This platform serves as a digital hub for parents, students, and staff of Rayalaseema High School.

![School Banner](https://images.unsplash.com/photo-1541339907198-e08756dedf3f?q=80&w=2070&auto=format&fit=crop)

## 🌟 Key Features

- **💎 Premium Design**: A modern, high-fidelity user interface built with a focus on typography, smooth animations, and a professional aesthetic.
- **🌐 Smart Language Toggle**: Custom-built Google Translate integration featuring a smart "EN/TE" toggle button with a pulsing attention ring for easy switching between English and Telugu.
- **📚 Digital Textbook Library**: Instant access to the official Andhra Pradesh State Syllabus (2024-25) textbooks for Classes 1 through 10.
- **📱 One-Click Communication**: 
  - **WhatsApp Integration**: Direct link for admissions and enquiries.
  - **Click-to-Call**: Immediate phone connection to school administration.
  - **Google Maps**: One-tap campus navigation.
- **⚡ Performance First**: Optimized for lightning-fast load times and 100% mobile responsiveness.
- **🛠️ Digital Architect Signature**: Custom developer branding and portfolio integration.

## 🚀 Tech Stack

- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS + Framer Motion (for premium animations)
- **Icons**: Lucide React
- **Translation**: Google Translate API (Custom Wrapper)
- **Deployment**: Vercel / Netlify ready

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ajaykumarreddy-k/School-Landing-Site.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   # or
   bun install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   # or
   bun dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📐 Project Structure

- `src/components/`: Modular React components (Hero, About, Materials, FloatingButtons, etc.)
- `src/assets/`: Branding assets and images.
- `index.html`: Optimized head with custom Google Translate hiding logic.
